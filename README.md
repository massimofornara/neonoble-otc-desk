# NeoNoble OTC Off-Ramp – Volumi >€100M/giorno

1 NENO = 1000 € fisso  
Off-ramp immediato in EUR su IBAN Unicredit o crypto (MEXC auto-invii)  
Pagamenti senza deposito cliente

## Setup
- `cd backend`
- `npm install`
- `npm start`
- Apri http://localhost:5000

## Funzioni
- Quotazione per NENO in EUR o crypto
- Pagamento immediato su IBAN Unicredit o wallet (MEXC API)
- Rilascio NENO dal Safe multi-sig

Pronto per 100M+ €/giorno su IBAN: IT22B0200822800000103317304
